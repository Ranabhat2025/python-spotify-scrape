# Spotify Billboard Top 100 Scraper

A Python project that creates a **Spotify playlist of the Billboard Hot 100 songs** for any date you choose. Simply enter a date, and the script fetches the top songs from that day and adds them to a private Spotify playlist.

---

## Features

* Scrapes Billboard Hot 100 songs for a specific date.
* Searches and adds songs to a Spotify playlist.
* Handles missing tracks gracefully.
* Private playlist creation on Spotify.

---

## Tech Stack

* Python
* Requests
* BeautifulSoup4
* Spotipy (Spotify API)

---

## Usage

1. Clone the repository:

```bash
git clone https://github.com/<your-username>/python-portfolio.git
```

2. Install dependencies:

```bash
pip install requests beautifulsoup4 spotipy
```

3. Update `main.py` with your Spotify Developer credentials and username.
4. Run the script:

```bash
python main.py
```

5. Enter the date in `YYYY-MM-DD` format when prompted.
